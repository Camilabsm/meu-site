Sobre mim

Olá! Me chamo Camila, tenho 28 anos e sou de Fortaleza - CE. Sou engenheira química de formação, porém, após alguns anos trabalhando na indústria, decidi fazer uma transição de carreira para a área de tecnologia.

Meu primeiro contato com a programação foi na faculdade, onde incorporei no meu TCC um algoritmo em Python que realizava o dimensionamento de um equipamento a partir de dados iniciais inputados pelo usuário. Hoje, sou mais familiarizada com HTML, CSS e JS, criando produtos responsivos de acordo com a necessidade de meus clientes.

Quando não estou escrevendo código ou assistindo aulas, estou assistindo séries, saindo para tomar um café ou jogando no meu Nintendo Switch.
